# Mathematical Feature Transformations using FunctionTransformer

This project explores **Mathematical Feature Transformations**, an important feature engineering technique used to reshape skewed numerical data into distributions that are easier for certain machine learning algorithms to learn from.

Using Scikit-Learn's **FunctionTransformer**, we apply common mathematical functions such as **Log**, **Reciprocal**, **Square**, and **Square Root** transformations to numerical features and analyze their impact on data distribution and model performance.

## 🎯 Key Takeaways

- Mathematical transformations help convert skewed data into a distribution that is often easier for machine learning models to learn.
- Always diagnose your data before applying transformations.
- Different transformations are suitable for different types of skewness.
- Tree-based models are generally insensitive to feature distributions, while many statistical and linear models benefit significantly from transformed features.
- FunctionTransformer provides a clean and reusable way to include custom mathematical transformations inside Scikit-Learn pipelines.
