# Deconstructing Unsupervised Learning: Discovering Patterns Without Labels

> *Part of my **Relearning ML in Public** series — revisiting Machine Learning from first principles and documenting the journey along the way.*

---

## 📖 About This Repository

Most resources on Machine Learning focus on **how to use algorithms**.

Far fewer explain **why those algorithms exist**, **what problems they solve**, or **how to build intuition before diving into the mathematics**.

This repository accompanies my three-part Medium series on **Unsupervised Learning**, where I revisit the subject from first principles.

Instead of treating unsupervised learning as just "machine learning without labels," the goal is to understand it as a completely different way of thinking about data—one centered around **discovery rather than prediction**.

The series gradually builds intuition before introducing the core concepts and algorithms, making it suitable for beginners as well as anyone revisiting the fundamentals.

# 🧠 Key Ideas Explored

Throughout this series, a few recurring themes appear again and again.

- Machine Learning is ultimately about learning **structure** from data.
- Supervised learning predicts.
- Unsupervised learning discovers.
- Geometry provides the language through which many algorithms understand similarity.
- Every algorithm makes assumptions about the data it receives.
- Understanding the intuition behind an algorithm is often more valuable than memorizing its implementation.
